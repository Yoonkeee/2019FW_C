# 2019FW_C
2019년도 2학기 시스템프로그래밍기초 (교수:강경태)
PBL 과제(무한 계산기) 코드

## <진행률>  
### 덧셈 90%  
음수+양수 덧셈 판단 남음  
### 뺄셈 90%  
음수 덧셈으로 넘어온 경우 연산 남음  
### 곱셈 0%  
### 후위연산 Stack 0%

## <수정>  
정수부 0으로 채우는 함수(fillZeroInt)와  
소수부 0으로 채우는 함수(fillZeroDec) 분리

## < 해야 되는 것들 >
곱셈 - 정우  
각 계산에 음수가 들어올 경우의 예외처리(ex: 1 - 110 = -109 ) - 민지  
후위표기 변환 & 스택 구현 - 지연
